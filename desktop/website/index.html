What is the Ozark STEM Interface?

The Ozark STEM Interface is a web portal that uses an algorithm to match STEM professionals with students. Read more

Register with the Ozark STEM Interface

How our Program Works

Our web-portal uses a mentor matching algorithm to match Teachers with STEM professionals, to share expertise. Our algorithm asks a bunch of questions and they are put into the algorithm to figure out the [best] possible candidate for matching. Our sophisticated algorithm will then match the Teacher, with the STEM professional.

Heavy Mettle Robotics Team

The Heavy Mettle Robotics Team (above), are the creators of this site!
They created it as a way for students to be able to get mentorship from local experts, because when they started out they needed help too.
